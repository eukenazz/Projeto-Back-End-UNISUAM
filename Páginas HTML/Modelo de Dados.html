Modelo de Dados!


Abaixo, o modelo de dados utilizado no projeto.

A concepção deste modelo de dados foca em ser simples e direto, onde há uma tabela de usuário que contém todas as suas informações relevantes e pertinentes como nome, data de nascimento, número de celular, etc. e uma para gravação de logs.

A tabela de logs obtém a id de usuário da tabela de usuário, para criar este relacionamento entre elas e assim obter suas informações. A ideia foi gravar informações importantes como eventos dentro do sistema e horários.
